Setup VPN clients

First, click on the power icon to download and install the correct app for your specific platform. If none of the links work, download and install manually from here. You can install the app on as many devices as you like, only the total bandwidth is limited.

  • Andriod/iOS
    1. Scan the QR code to add your profile
    2. Make sure to activate the VPN mode
    3. In the config options menu, update these settings:
      • Remote DNS = tcp://8.8.8.8
      • Strategy = ipv4_only
      • Direct DNS = 8.8.8.8
      • Direct Strategy = ipv4_only
    4. Use the per-app proxy settings to allow apps like Taobao to bypass the VPN
  • Windows
    1. After installing, open up the installation folder
    2. Right-click Hiddify.exe, select Properties -> Compatibility, then check 'Run as Administrator'
    3. To add your profile, copy the shared link and import from clipboard
    4. Make sure to activate the VPN mode
